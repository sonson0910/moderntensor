# Developer Guide

Welcome to the ModernTensor Developer Guide.

Here you will learn how to build decentralized AI applications (dApps) on top of the ModernTensor blockchain.

## Getting Started

1. **Smart Contracts**: Learn how to call AI models from Solidity.
2. **SDK**: Interact with the blockchain using Python using `mtcli` or `moderntensor-sdk`.
3. **RPC**: Deep dive into the JSON-RPC API.
