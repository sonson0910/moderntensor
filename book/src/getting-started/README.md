# Getting Started

This section will guide you through setting up a LuxTensor node and interacting with the network.

## Prerequisites

- **OS**: Linux (Ubuntu 22.04+ recommended), macOS, or Windows (WSL2).
- **Rust**: Version 1.75 or higher.
- **Hardware**:
  - CPU: 4+ cores
  - RAM: 8GB+ (16GB recommended)
  - Storage: 100GB+ SSD

## Quick Steps

1. **Install Rust**: `rustup update`
2. **Clone Repo**: `git clone https://github.com/sonson0910/luxtensor`
3. **Build**: `cargo build --release`
4. **Run**: `./target/release/luxtensor-node`
